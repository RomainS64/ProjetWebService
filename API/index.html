GET


SunnyPath c'est quoi?

SunnyPath est un webservice qui permet aux habitants de la côte Basque de connaître le nombre de personnes allant aux même endroit qu'eux en utiisant le même moyen de locomotion. 

      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php
    
  


      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php?locomotion=MoyenDeLocomotion

MoyenDeLocomotion: velo/bus/voiture

Retourne le nombre de personnes ayant pris le Moyen De Locomotion dans la journée. 

      
      {
        "result":
        "78"
      }
    
  


      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php?ville=Ville

Ville: ville en minuscule

Retourne le nombre de personnes ayant voyagé dans la Ville dans la journée. 

      
      {
        "result":
        "24"
      }
    
  


      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php?horaire=Horaire

Horaire: horaire de voyage (ex: 15 pour 15h-16H)

Retourne le nombre de personnes ayant voyagé dans l'Horaire cité, dans la journée. 

      
      {
        "result":
        "8"
      }


Les options Horaire, MoyenDeLocomotion et Ville sont cumulables

Par exemple: 

      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php?horaire=Horaire&ville=Ville

Va renvoyer le nombre de voyageurs de la ville Ville à l'horaire Horaire 

      
      {
        "result":
        "14"
      }
    
  


      
      http://iparla.iutbayonne.univ-pau.fr/~rsalha/WebService/projet/API/GET.php?horaire=Horaire&ville=Ville&locomotion=MoyenDeLocomotion

Va renvoyer le nombre de voyageurs de la ville Ville à l'horaire Horaire utilisant le moyen de locomotion MoyenDeLocomotion 

      
      {
        "result":
        "3"
      }
